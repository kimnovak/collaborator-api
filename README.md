# collaborator-api

API for the collaborator project https://github.com/q-praksa/collaborator 

## Prerequisites
`node >= 14`  
  
## Install
`npm i -g sequelize-cli`  
`npm i -g sequelize-mig`  
`npm i -g nodemon`  
`npm i`  

## Usage
`npm run dev`

## Migrating DB
`npm run make-migration`  
`npm run migrate`
